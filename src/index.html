RYBitten

Implementation of pseudo RYB color conversions derived from Johannes Itten's color wheel.

RYB Colorspace

Trilinear interpolation is the method used by this library to translate between RGB and RYB color spaces, creating an imaginative new color space that aligns with RGB.

The RGB/RYB cube has been manually tuned to emulate Johannes Itten's chromatic circle. By manipulating the cube's edges, you can see how the colors change and create a colorspace that fits your artistic needs.
